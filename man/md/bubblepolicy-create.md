# bubblepolicy-create(1) -- Create a bubblewrap wrapper

## SYNOPSIS

**bubblepolicy create** [_POLICY_]

## DESCRIPTION

Create a bubblewrap wrapper script from a policy file. Outputs to stdout.

## OPTIONS

_POLICY_ - Policy file (default: policy.txt).

## EXAMPLE

    $ bubblepolicy create policy.txt > wrapper.sh
