# bubblepolicy-optimise(1) -- Optimise/dedup the policy tree

## SYNOPSIS

**bubblepolicy optimise** _FILE_

## DESCRIPTION

Optimise and deduplicate the policy tree in place. Collapses paths with identical access into parent directories.

## OPTIONS

_FILE_ - Input/output file (required).

## EXAMPLE

    $ bubblepolicy optimise policy.txt
